## 🚨 常见问题解决

### 更新后网站没有变化？
1. 检查部署平台的状态页面
2. 查看构建日志是否有错误
3. 清除浏览器缓存
4. 等待1-3分钟完成部署

### 交易对数据不显示？
1. 检查数据库连接配置
2. 验证环境变量设置
3. 查看API端点是否可访问
4. 检查控制台错误信息

### 性能问题？
1. 检查数据库索引是否创建
2. 验证API响应时间
3. 查看是否有大量数据加载
4. 考虑启用缓存机制
